## Overview

Arbi AI helps you streamline your hiring process through AI-powered candidate screening and evaluation. This guide will help you set up your first project and start evaluating candidates.

## Creating Your First Project

### 1. Project Setup

Start by creating a new project to evaluate candidates for a specific position:

1. Click "New Project" from your dashboard
2. Enter your project details:
   - Project title
   - Position description
   - Select data type(s):
     - Documents (Resume DOC/PDF)
     - Audio
     - Video
     - Mixed (Multimodal)

<Note>
  Choose the data type based on what you'll be evaluating - resumes, interviews, or multimedia submissions.
</Note>

### 2. Configure Project Properties

<AccordionGroup>
  <Accordion icon="users" title="Set Project Access">
    Define who can access and manage your project:
    - **Project Owner/Manager**: Full control over project settings and evaluations
    - **Evaluator**: Can review and assess candidates
    - **Viewer**: View-only access to project data
  </Accordion>
  
  <Accordion icon="calendar" title="Set Timeline">
    Configure your project timeline:
    - Set maximum number of submissions
    - Define submission deadline (or select "no deadline")
    - Schedule evaluation periods
  </Accordion>
</AccordionGroup>

## Setting Up Your Evaluation Rubric

The rubric is crucial for AI-powered candidate assessment. Define your criteria based on:

<CardGroup>
  <Card title="Experience & Proficiency" icon="briefcase">
    Specify required years of experience, technical skills, and expertise levels
  </Card>
  
  <Card title="Education & Training" icon="graduation-cap">
    Define educational requirements and necessary certifications
  </Card>
  
  <Card title="Custom Criteria" icon="list-check">
    Add position-specific evaluation criteria with customizable scoring scales
  </Card>
</CardGroup>

## Workspace Organization

Your workspace helps you manage multiple projects efficiently:

- Organize by creator/department
- Group related submissions
- Sort by evaluation dates
- Track project status

<Note>
  Use separate directories to maintain clear organization and streamline project management.
</Note>

## Assessment Types

Choose from several assessment types based on your needs:

- AI general evaluation
- Resume screening
- Customer service call review
- Tradewinds CDAO Pitch Analysis
